## 기존 방식의 함수 정의와 사용
먼저 기존 방식으로 함수를 정의하고 사용하는 과정을 살펴보자.  
다음은 중학교 때 배운 함수식이다.

$y = f(x) = 3 \times x + 1$

이 식에서  
$x$가 1일 때 $y = f(1) = 3 \times 1 + 1$이 되어 $y$는 4가 된다.  
$x$가 2일 때 $y = f(2) = 3 \times 2 + 1$이 되어 $y$는 7가 된다.  
$x$가 -1일 때 $y = f(-1) = 3 \times (-1) + 1$이 되어 $y$는 -2가 된다.

### 함수 정의하고 사용해 보기
이제 기존 방식으로 함수 f를 정의하고 사용해 보자.
```python
def f(x):
    return 3*x + 1

x = 10
y = f(x)

print(f' y: {y}')
```
```
 y: 31
```
