## 다양한 인공 신경망 구현해 보기
우리는 앞에서 입력1 출력1로 구성된 인공 신경의 동작을 살펴보고 구현해 보았다.  
여기서는 입력2 출력1의 단일 인공 신경과, 입력2 출력2, 입력3 출력3으로 구성된 인공 신경망의 구조를 살펴보고 수식을 세운 후, 해당 수식에 맞는 인공 신경망을 구현해 보겠다.
